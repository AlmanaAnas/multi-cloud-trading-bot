Security

Zero credentials stored anywhere

Every auth boundary uses short-lived tokens. No access keys, no service account files, no passwords.

🔑
GitHub → AWS
OIDC token assumes an IAM role locked to this exact repository. No AWS_ACCESS_KEY_ID ever stored.
✓ keyless CI auth
🌉
GitHub → GCP
Workload Identity Federation maps GitHub's OIDC token to a GCP service account. Attribute condition locks it to this repo.
✓ no SA key file
⚡
Lambda → Pub/Sub
Lambda exchanges its AWS identity for a 1-hour GCP token at runtime via WIF. Nothing stored anywhere.
✓ cross-cloud keyless
🛡️
Private networking
Lambda in a private subnet with VPC endpoints. Cloud Function behind firewall rules. No unnecessary public exposure.
✓ VPC + security groups
🗄️
Encrypted state
Terraform state encrypted at rest in S3 (AES-256). Public access blocked. DynamoDB prevents concurrent corruption.
✓ encrypted + locked
💣
One-click teardown
workflow_dispatch GitHub Action destroys all 41 resources in both clouds. No orphaned resources, no surprise costs.
✓ terraform destroy
